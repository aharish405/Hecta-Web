📋 Quick Instructions

  1. Design your icon using one of the tools below
  2. Use these colors:
    • Background: Dark Blue (#1e3a5f) to Navy (#0a1628)
    • Letter/Logo: Orange (#ff6b35) gradient
  3. Create two sizes:
    • 192 x 192 pixels (save as: icon-192x192.png)
    • 512 x 512 pixels (save as: icon-512x512.png)
  4. Save to: public/ folder in your project
  5. Create favicon: 32x32 pixels (save as: favicon.ico)

💡 Design Tips

  • Keep it simple - the icon will be small on mobile devices
  • Use high contrast for visibility
  • Consider a rounded square shape (not perfect circle)
  • Test on both light and dark backgrounds
  • The letter "H" is a good starting point for Hecta
  • Add subtle metallic or gradient effects for premium look

✅ Website works without icons!

The website is fully functional without custom icons. They just enhance the PWA experience when users install the app on their devices.
